DDNS 设置

多服务商内置 DDNS；也支持切到 ddns-go 原生模式（全服务商）。配置写入 ddns_config.json。需 pip install tldextract 以精准解析多级域名。

总开关
公网地址获取

与 ddns-go 类似：URL 多行探测，或 interface 填 Linux 网卡名（如 wlo1）。

要更新的 FQDN

每行一个完整主机名，例如 home.example.com。IPv4 → A 记录，IPv6 → AAAA 记录。

TrafficRoute（火山引擎）AK / SK

在火山引擎控制台创建 AccessKey，授权 TrafficRoute 写权限。Secret 仅在局域网下使用。

CloudflareAPI Token

推荐只填 API Token；Zone ID / Record ID 留空时自动查询（需 Token 有 Zone.DNS 写权限）。

阿里云 DNSRAM AK

RAM 控制台创建子账号，授权 AliyunDNSFullAccess，取得 AccessKey。

DNSPod（腾讯云）API Token

在 DNSPod 控制台 创建 API Token，格式为 ID,Token。
